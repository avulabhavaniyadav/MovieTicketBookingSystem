# Movie Ticket Booking System

## Project Title
Movie Ticket Booking System using Java

---

## Project Description
The Movie Ticket Booking System is a Java-based application developed to automate the process of booking movie tickets in a theatre. The system helps users select seats, book tickets, and simulate payment processing efficiently.

This project is designed using Java concepts such as JDBC, Collections Framework, Exception Handling, and Multithreading. It prevents duplicate bookings and maintains booking details using a MySQL database.

---

## Features
- Seat Selection
- Ticket Booking
- Payment Simulation
- Prevent Duplicate Booking
- Real-time Seat Availability
- JDBC Database Connectivity
- Multithreading Support
- Exception Handling

---

## Technologies Used
- Java

- Collections Framework

## Steps to Run the Project

1. Install Java and MySQL.
2. Create database in MySQL.
3. Open project in VS Code.
4. Add JDBC Driver.
5. Compile Java files.
6. Run MovieTicketBooking.java.
7. Enter customer details and seat number.
8. Booking details will be stored in database.
