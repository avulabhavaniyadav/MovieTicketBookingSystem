# Movie Ticket Booking System

## Project Description
The Movie Ticket Booking System is a Java-based application developed to automate the process of booking movie tickets. The system allows users to select seats, book tickets, and simulate payment processing. It uses Java concepts like JDBC, Collections, and Multithreading.

## Features
- Seat Selection
- Ticket Booking
- Payment Simulation
- JDBC Connectivity
- Multithreading Support
- Prevent Duplicate Booking

## Software Requirements
- JDK 8 or above
- MySQL Database
- VS Code or IntelliJ IDEA
- MySQL JDBC Driver

## Steps to Run the Project
1. Install Java and MySQL.
2. Create database using SQL queries.
3. Open project in VS Code.
4. Add MySQL JDBC Driver.
5. Compile Java files.
6. Run MovieTicketBooking.java.
